## Final Project for DATA 35900 (Spring 2026)


### Description:
This project studies privacy failure in local differential privacy through the lens of statistical physics. The central question is whether adversarial inference changes smoothly as the privacy budget increases, or whether high-dimensional aggregation can produce a phase-transition-like shift from a protected regime to an exposed regime. We simulate sparse support recovery under randomized response, where a hidden set of active coordinates must be inferred from locally privatized binary observations. The simulation measures recovery overlap and posterior entropy collapse as order parameters, then tests whether these quantities organize around the scaling variable \(\lambda = n a^2 \tanh^2(\epsilon/2)/\log d\). This allows us to compare exact recovery with broader uncertainty collapse and evaluate whether privacy loss begins before perfect reconstruction succeeds.
